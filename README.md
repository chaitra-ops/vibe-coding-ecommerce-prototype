# Fruit Shop – Vibe Coding E-Commerce Prototype

A responsive e-commerce web application built using **GitHub Copilot Agent** and the **Vibe Coding** workflow as part of the Microsoft Learn **"Introduction to Vibe Coding"** module.

## 🚀 Project Overview

This project demonstrates how AI-assisted development can accelerate the software development lifecycle by transforming natural language requirements into a functional application.

The application was created using a structured vibe coding process:

* Product Vision
* Product Requirements Document (PRD)
* Wireframe Design
* AI-Assisted Code Generation
* Iterative Development and Refinement

## ✨ Features

### 🛍️ Shopping Experience

* Product listing page
* Product search functionality
* Product details view
* Add products to cart
* Remove products from cart
* Quantity management
* Shopping cart summary
* Checkout process
* Order confirmation / Thank You page

### 🎨 User Interface

* Responsive design
* Modern e-commerce layout
* Interactive product cards
* Cart item count badge
* Toast notifications
* Mobile-friendly navigation

## 🛠️ Technologies Used
